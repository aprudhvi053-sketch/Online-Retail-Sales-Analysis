# Online Retail Sales Analysis
## Project Overview
This project analyzes an online retail dataset using Power BI to uncover insights about sales trends, product performance, and regional demand.

## Dataset
The dataset contains transactional sales data including:
- Invoice ID
- Product
- Quantity
- Price
- Country
- Date

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

## Dashboard Features
- Revenue trend over time
- Top-selling products
- Sales by country
- Customer purchasing patterns

## Key Insights
- United Kingdom generates the highest revenue
- Seasonal spikes in sales during holiday periods
- Few products contribute majority of revenue
